-   HTML 태그의 기본 속성에 대해서 알아야 합니다.
    -   Block, Inline, Inline-Block, Flex
    -   li, div, span, table
-   글자색, 배경색과 같은 기본 CSS를 사용 할줄 알아야합니다.

    -   relative
    -   static
    -   absolute
-   dom selector가 무엇 인지 알아야합니다.

    -   Transition
    -   KeyFrame
-   아래 Css가 무엇인지 알아야 합니다.
-   html 파일에 Css를 링크 거는 법을 알아야 합니다.
