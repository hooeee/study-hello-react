## 개발 환경 셋팅
1. Nodejs 다운로드 사이트에서 다운로드
- https://nodejs.org/ko/

2. vscode 에디터 다운로드
- https://code.visualstudio.com/

3. 프로젝트 생성 
- 작업용 폴더 생성
- 폴더 내에서 터미널을 통해 명령어 입력
```cmd
npx create-react-app [프로젝트이름]
```

4. app 실행 
```cmd
npm start
```

## 파일 설명
#### node_modules
- 라이브러리 코드 보관함

#### public
- static 파일 보관함

#### src
- 소스 코드 보관함
- app.js -> 메인

#### package.json
- 프로젝트 정보
- 라이브러리 정보
