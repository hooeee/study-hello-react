## 03. 코드 길어지면 import export 하면 됩니다

- Data.js 파일 만들어주고

- import 해서 App.js에서 써보자.

  - Data.js

    ```react
    let a = 10;
    
    export default a;
    ```

  - App.js

    ```react
    import a from './Data.js';
    
    {a}
    ```

- 두개이상 쓰고 싶은 경우

  ```react
  // Data.js
  let a = 10;
  let b = 100;
  
  export default {a, b};
  
  // App.js
  import {a,b} from './Data.js';
  ```

  - {} 중괄호 가져올때 같은 이름써야함, 구조분해 할당이라서
  - 변수 뿐아니라 , 함수, 컴포넌트 등도 가능함

- 실습을 위한 Data.js
